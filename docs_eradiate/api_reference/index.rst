API Reference
=============

This page documents the Python-facing API introduced by the Eradiate kernel. Only additions specific to this fork are covered here; the base Mitsuba 3 API is
documented in the
`Mitsuba 3 documentation <https://mitsuba.readthedocs.io/en/stable/src/api_reference.html>`__.

.. dropdown:: How to build this page
   :color: muted
   :icon: alert

   This page requires two steps before it renders correctly:

   1. Build the sources.
   2. Build the API subsite to generate the intermediate RST.

   These steps are automated by the ``pixi run docs-api`` task. Then rebuild this site normally.
